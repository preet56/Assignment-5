How to Apply CSS to HTML

An external stylesheet contains CSS in a separate file with a .css extension. You reference an external CSS stylesheet from an HTML <link> element. The href attribute of the <link> element needs to reference the location of the CSS file on your file system. Here's an example of how to add an external CSS file to your HTML document: 

        
            <head>
                <link rel="stylesheet" href="styles.css" />
            </head>

How to Use JavaScript in a Webpage

To include JavaScript in an HTML document, you can either link to an external JavaScript file or embed JavaScript code directly within the HTML file. Below are examples of both methods: 

    
        // Linking to an external JavaScript file
        <script src="script.js"></script>

        // Embedding JavaScript code within an HTML file
        <script>
            console.log('Hello, world!');
        </script>
